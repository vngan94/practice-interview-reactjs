  • What is virtual DOM? How does it work?
    • Purpose: make UI updates faster and more efficient.
    • is lightweight copy of the real DOM
    • How does it work?
      • UI is first built using the virtual DOM.
      • When data changes, a new virtual DOM is created.
      • React compares it to the old virtual DOM (diffing).
      • It finds the smallest changes needed.
      • Only those updates are applied to the real DOM (reconciliation).
